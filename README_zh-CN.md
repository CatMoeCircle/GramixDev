## 注意

本项目处于开发阶段

## 描述

QTalkgram是一个使用Electron+Vue3+TDLib的第三方客户端

## 关于

大家可以通过我们的交流群 https://t.me/QTalkgramGroup 来获取最新的开发进度以及与我联系。

## 调试

```bash
# 克隆本项目并进入目录
npm install
npm run dev
```
